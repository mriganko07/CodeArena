# 🚀 How to Run CodeArena

## 📦 Install Dependencies

### Root

```bash
npm install
```

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Project

Go to **root folder** and run:

```bash
npm run dev
```

---

## 🌐 Ports

* Frontend → http://localhost:5173
* Backend → http://localhost:4000
